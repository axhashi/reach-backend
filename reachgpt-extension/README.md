# ReachGPT Chrome Extension

LinkedIn stalking, but make it sales.

## Install in 60 seconds

1. Open Chrome and go to: `chrome://extensions`
2. Toggle **Developer mode** ON (top right corner)
3. Click **Load unpacked**
4. Select this folder (`reachgpt-extension`)
5. The ReachGPT icon appears in your toolbar ✓

## How to use

1. Go to any LinkedIn profile: `linkedin.com/in/anyprofile`
2. Click the ReachGPT extension icon
3. It reads the page — name, headline, recent posts
4. Type your pitch (what you're selling)
5. Enter your Anthropic API key (get one at console.anthropic.com)
6. Hit Generate → copy the email → send it

## Get your API key

1. Go to https://console.anthropic.com
2. Create account → API Keys → Create Key
3. Paste it into the extension (saved locally, never leaves your browser)

## Notes

- Works on ANY public LinkedIn profile
- Uses real data from the page — no hallucinated signals
- API key is stored locally in Chrome storage
- Cost: ~$0.01 per email generated
